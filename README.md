### General installation guides

#### Step 1: Clone the repository

```shell
git clone https://github.com/ayodeleoniosun/products-restful-api.git
```

#### Step 2: Switch to the repo folder

```shell
cd products-restful-api
```

#### Step 3: Dockerize app

```shell
bash setup.sh
```
